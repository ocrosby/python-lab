# Core Scientific & Statistical Libraries

## NumPy

- Foundation for all numerical computing in Python
- Fast vectorized operations, arrays, linear algebra

## SciPy

- Advanced statistics, optimization, signal processing
- Probability Distributions, Hypothosis Tests, Integration

## Pandas

- Tabular data analysis
- Powerful grouping, merging, windowing, time series


## Polars

- Extremely fast DataFrame library (Rust backend)
- Lazy evaluation, multithreaded, arrow-native
- Usually faster than Pandas for large datasets

## MatplotLib / Seaborn

- Foundation for statistical plots

## Plotly

- Interactive charts, dashboards

## Altair

- Declarative visualization, great for statistical graphics
